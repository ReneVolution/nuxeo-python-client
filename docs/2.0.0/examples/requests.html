Run requests¶

With nuxeo.request(...) you can run any request you want on the REST API. The following list of examples isn’t exhaustive, but you can search for available operations on the explorer (click on Search Operations on the version corresponding to your Nuxeo Platform).

Fetch the Administrator user

user = nuxeo.client.request('GET', 'user/Administrator')

Fetch the whole list of Natures

natures = nuxeo.client.request('GET', 'directory/nature')
